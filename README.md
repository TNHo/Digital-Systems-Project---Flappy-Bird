# Digital Systems Project - Flappy Bird
RTL Design - Flappy Bird

## Summary
This project implements Flappy Bird as a custom processor in VHDL. 

![HLSM](https://raw.githubusercontent.com/TNHo/Digital-Systems-Project---Flappy-Bird/refs/heads/main/Report/Flappy_Bird_HLSM.png)

We do this by taking a 'High Level State Machine' diagram and translate its behavhiors into multiple circuits.
